# Creator Imagery Prompts for WAI Institute

Generate these images using DALL-E, Midjourney, or Stable Diffusion. All images should:
- Feature Black creators/artists authentically
- Use warm, inviting photography/illustration style
- Reflect diverse expressions of creativity
- Maintain professional, dignified representation
- 16:9 aspect ratio recommended

## Installation Instructions

1. Generate images using your preferred AI service
2. Save as `.jpg` or `.webp` in `frontend/public/images/creators/`
3. Name files: `creator-1-poet.jpg`, `creator-2-artist.jpg`, etc.
4. Update Landing.jsx with image paths as shown in the code update

## Recommended Tool
- **DALL-E 3** (highest quality for professional representation)
- **Midjourney** (excellent for artistic/authentic feel)
- **Stable Diffusion XL** (free option with good results)

Use same seed and style parameters across images for visual consistency.
